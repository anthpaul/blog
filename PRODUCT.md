# Product

## Register

brand

## Users

Students and practitioners studying Linux/Ubuntu system security. Academic context — likely following a course structure. They read articles sequentially (the series), copy terminal commands, and reference security tools like Lynis and CIS Benchmarks.

## Product Purpose

SecUbuntU is an academic blog covering Ubuntu system security: fundamentals, vulnerability analysis, hardening, and best practices for Ubuntu 22.04 / 24.04 LTS. Success = readers completing the series with enough knowledge to harden a real system.

## Brand Personality

Technical, trustworthy, direct. Ubuntu-native (orange brand, Ubuntu font), not generic "dark hacker" aesthetic. Feels like authoritative documentation with a human voice.

## Anti-references

- Generic "hacker" dark-mode blogs with neon green on black
- Overcrowded security wikis with no visual hierarchy
- SaaS-cream editorial blogs (warm neutral body background)

## Design Principles

1. **Content first** — chrome and decoration exist only to serve the article text
2. **Terminal-native accents** — code blocks, monospace labels, and CLI snippets are first-class UI citizens
3. **Sequential clarity** — the series structure must always be navigable from any page
4. **Earned restraint** — brand orange appears only where it carries meaning, not decoration

## Accessibility & Inclusion

WCAG AA target. Spanish-language content; all UI copy in Spanish. No specific reduced-motion or color-blindness requirements stated, but standard WCAG AA contrast compliance is required.
